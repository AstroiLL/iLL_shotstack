## Why

Текущая утилита проверки JSON (`check_json.py`) не обеспечивает полноценную валидацию Shotstack скриптов, что приводит к ошибкам сборки видео и необходимости ручной отладки. Необходима надежная проверка на всех этапах рабочего процесса.

## What Changes

- Улучшенная валидация JSON структуры Shotstack Template
- Проверка доступности медиафайлов перед сборкой
- Валидация полей в соответствии с Shotstack SDK
- Интеграция проверки в конвертер и сборщик видео
- Унификация процесса проверки для всех рабочих процессов

## Capabilities

### New Capabilities
- `json-validation`: Комплексная валидация Shotstack Template JSON структуры
- `media-file-availability`: Проверка доступности медиафайлов в локальной файловой системе
- `sdk-field-validation`: Валидация полей согласно Shotstack API спецификациям
- `integrated-validation`: Автоматическая проверка в конвертере и сборщике

### Modified Capabilities
- Отсутствуют - это новое изменение не модифицирует существующие спецификации

## Impact

Затронутые компоненты:
- `check_json.py` - основная утилита валидации
- `convert_script.py` - интеграция проверки перед конвертацией
- `assemble.py` - интеграция проверки перед сборкой видео
- `fast_clip/check/` - модуль валидации
- `fast_clip/assembler.py` - основной оркестратор сборки

Внешние зависимости:
- Shotstack API документация для валидации полей
- Pydantic для структурированной валидации
- Модули файловой системы для проверки доступности файлов
