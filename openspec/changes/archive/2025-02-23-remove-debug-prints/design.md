## Контекст

В методе assemble_from_template() класса VideoAssembler есть отладочный print-оператор, который используется во время разработки.

## Цели / Не цели

**Цели:**
- Удалить отладочный print-оператор, сохраняя остальную функциональность verbose-режима

**Не цели:**
- Изменение логики verbose-режима
- Удаление других print-операторов (они являются частью пользовательского интерфейса CLI)

## Решения

### Решение 1: Простое удаление строки

Удалить строку 424 с print(f"Debug: Using resources_dir = {resources_dir_name}") так как это отладочный артефакт, а не функциональность.
